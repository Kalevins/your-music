# Your Music

_Proyecto que consume la API de Spotify para mostrar las canciones de un artista y guardarlas en una lista de favoritos._

### 🔧 Instalación

_En el directorio del proyecto, puede ejecutar:_

```
npm i
```
```
npm run dev
```

_Para correr las pruebas:_

```
npm run test
```
