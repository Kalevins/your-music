# Your Music

_Prueba técnica para 57Blocks la cual consume la API de Spotify.._

### 🔧 Instalación

_En el directorio del proyecto, puede ejecutar:_
```
npm i
```
```
npm run dev
```

_Para correr las pruebas:_
```
npm run test
```
